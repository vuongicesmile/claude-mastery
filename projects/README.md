# Projects — Apply Everything

## Project 1: AI-Powered Code Reviewer CLI
```
Build a CLI tool that:
- Takes a git diff as input
- Uses Claude API to review for bugs + security
- Outputs structured findings with severity
- Can post review as GitHub PR comment (MCP)
- Uses prompt caching for system prompt

Skills used: API, Tool Use, MCP, Prompt Engineering
```

## Project 2: Personal Dev Assistant with Memory
```
Build an assistant that:
- Remembers your codebase conventions
- Learns from your corrections
- Answers questions about your code
- Uses Files API for large codebases
- Maintains conversation history

Skills used: API, Files API, Extended Thinking, Prompt Caching
```

## Project 3: Automated PR Review Workflow
```
GitHub Actions workflow that:
- Triggers on PR open/update
- Spawns Claude agents for different review dimensions
- Aggregates findings
- Posts inline comments via GitHub MCP
- Assigns severity labels automatically

Skills used: Workflows, Agent Patterns, MCP, GitHub Actions
```
